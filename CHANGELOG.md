# Changelog

All notable changes to FrameFeed are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/) and releases use semantic
versioning.

## [0.1.0] - 2026-07-21

### Added

- Local folder and experimental Apple Shared Album sources.
- Deterministic recency-weighted rotation and one-hour new-photo feature window.
- Local YuNet face detection, face-safe crops, and blurred contain fallback.
- Private JSONL analysis cache and last-good-frame behavior.
- Hardened Docker Compose deployment with Unraid and reTerminal E1004 guides.
- Landscape and portrait E1004 presets.
